# Imágenes Necesarias

Coloca aquí las siguientes imágenes:

## Requeridas
- `logo.png` - Logo principal del sitio (recomendado: 320x100px)
- `loading.gif` - GIF de animación de carga (recomendado: 80x80px)

## Opcionales (para fondos de botones)
Si no existen, los botones mostrarán el color verde sólido.

- `fondo.jpg` - Imagen de fondo general (opcional)
- `btn-apuestas.jpg` - Fondo del botón "Mis Apuestas"
- `btn-clasificacion.jpg` - Fondo del botón "Clasificación"
- `btn-canjear.jpg` - Fondo del botón "Canjear Puntos"
- `btn-casino.jpg` - Fondo del botón "Casino"
- `btn-realizar.jpg` - Fondo del botón "Realizar Apuesta"
- `btn-historial.jpg` - Fondo del botón "Ver Historial"
- `btn-pendientes.jpg` - Fondo del botón "Jornadas Pendientes"
- `btn-jugadores.jpg` - Fondo del botón "Clasificación Jugadores"
- `btn-liga.jpg` - Fondo del botón "Clasificación La Liga"

## Formato recomendado
- PNG para logos (transparencia)
- JPG para fondos (menor tamaño)
- GIF para animaciones
